What it is
First protoype for revisions and improvements to the existing Fraud Referral Capture form. This prototype has been created to test a new start page and some knockout questions/guidance content at the start of the journey along with revised content on the confirmation page at the end of the journey. This new content essentially 'bookends' the start and end of the existing FRC form which has been 'scraped' directly from the live service.
Hypotheses to test
Because many FRC Referrals are duplicates, and some relate to fraud types handled by other government departments and agencies, or by authorities in Scotland and Northern Ireland, we believe that providing clearer guidance and redirection at the start of the referral journey will help discourage users from submitting repeat referrals. We will know this is effective when we see a reduction in the number of duplicate referrals received by RMS.
